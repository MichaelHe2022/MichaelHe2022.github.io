Project: Medieval Shootout

Coded in C, this is a turn-based game that deals with interaction between two players, using Trivia C Launchpads as controllers. The concepts that it implements are basic LED color, controller button interaction, and the transferring of data between two launchpads. Check out the code and implementation by clicking the image, which links to my GitHub.

Project: Sokoban

A version of the game Sokoban. In this game, the player attempts to push boxes around a warehouse to try to get them to the proper storage location. I have created my own maps and symbols representing the player, the box, the box target location, and wall characters. All of these are declared in a config file. The rest of the rules and conditions regarding player mechanics are explained within the method headers of the methods. Click on the image link to see the code!
